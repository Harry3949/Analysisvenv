# 🎧 Audio Analysis Lab

音楽ファイルをAIで精密に解析し、その特徴を可視化・言語化する「未来の音楽実験室」プラットフォームです。
ただの解析ツールではなく、ラボの案内役であるキャラクター「ノヴァ」と「ルミ」との対話を通じた音楽性診断など、エンターテインメント性を融合させた体験を提供します。

---

## ✨ 主な機能

1. **AIオーディオ解析**
   - `librosa` を活用し、BPM・キー・周波数バランス（Low/Mid/High）をリアルタイムに算出
   - 解析データに基づき、1,200通り以上のバリエーションを持つAIコメントを自動生成

2. **リアルタイム・ビジュアライザー**
   - Web Audio API を使用し、楽曲の波形と周波数スペクトラムをリアルタイムに描写

3. **AI音楽性診断**
   - キャラクター（ノヴァ・ルミ）とのチャット形式による対話型クイズ
   - ユーザーの感性を深層心理から紐解き、タイプ別の音楽性を判定
   - ログインユーザーの名前を呼びながら個別コメントを生成

4. **ユーザープロフィール＆履歴管理**
   - 解析履歴の自動保存・削除
   - プロフィールアイコンのカスタマイズ

## 🚀 起動方法

### 手順1：仮想環境の作成とライブラリのインストール

プロジェクトフォルダに移動後、以下を順番に実行してください。

```bash
# 仮想環境を作成
python -m venv venv

# 仮想環境を有効化
# Windowsの場合:
.\venv\Scripts\activate
# Mac/Linuxの場合:
source venv/bin/activate

# 依存ライブラリのインストール
pip install -r requirements.txt
```

### 手順2：アプリケーションの起動

```bash
python app.py
```

起動後、ブラウザで `http://127.0.0.1:8000` にアクセスしてください。

## 👨‍💻 開発のこだわり

- **エンターテインメント × 技術の融合**: 解析ツールにキャラクター演出と音楽性診断を組み合わせることで、技術的な機能に「体験」としての価値を加えました。
- **AIコメント生成**: 周波数・テンポ・ムードの組み合わせによる1,200通り以上のコメントバリエーションを実装し、毎回異なる分析体験を提供します。
- **パーソナライズ**: 診断の全工程でログインユーザー名を呼びかけ、個別コメントを添えることで、親密さと没入感を演出しています。
- **コードの可読性**: HTML / CSS / JavaScript を完全に分離し、保守性の高いファイル構成を採用しています。
